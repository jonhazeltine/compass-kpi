Algorithm Reference

Formula per KPI event:

initialPC = avg_price × commission_rate × (pc_weight / 100)

Phase 1: DELAY (0 → delay_days)
  value = $0

Phase 2: HOLD (delay → delay + hold)
  value = initialPC (full)

Phase 3: DECAY (hold_end → + decay_days)
  value = initialPC × (1 - elapsed/decay)

Phase 4: EXPIRED
  value = $0

TTC (Time to Close) = delay + hold days. Represents when this activity typically converts to a closing.

PC Weight = what fraction of a deal this activity represents. A "Listing Taken" (7.0) is worth 280x more than a "Phone Call" (0.025).

Decay is linear. Value drops to $0 over decay_days after the hold period ends.
