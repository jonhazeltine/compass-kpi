CSS Only

What you saw in v2. Gradients, pseudo-elements, inline SVG patterns, CSS keyframes.

  • Zero assets, zero load time
  • Works offline instantly
  • AI generates a theme as a JSON object
  • Limited to what CSS can draw
  • No photorealism, no complex illustrations
BASELINE — ALWAYS SHIP THIS

AI Background

Real photo/AI-generated texture as card background, darkened overlay keeps icon readable.

  • Real texture — frost, bokeh, pumpkins, sand
  • ~15KB per image (200x200 WebP)
  • Unique per challenge (AI generates on create)
  • Static — no motion
  • Need dark overlay to keep icon contrast
UPGRADE — PER CHALLENGE

Lottie Overlay

Animated particles (snow, hearts, smoke, waves) running inside each card.

  • 60fps smooth animation
  • ~5-20KB per Lottie JSON
  • Thousands of free seasonal Lotties available
  • 4 Lotties per card row = perf cost
  • Most alive, most premium feel
PREMIUM — HIGH-VALUE THEMES

They're not mutually exclusive. The card component takes a theme prop. A theme can specify CSS-only styling, an AI background URL, a Lottie overlay URL, or all three layered together. Ship CSS as the default, upgrade individual challenges to AI/Lottie when they deserve it.
