The point: The icon never changes — it's always a Phosphor SVG component with a color prop. The "wrapper" is just CSS: background gradient, border color, shadow color, glow filter, optional pseudo-element overlays (snow, fire, frost).

A challenge theme is literally a small JSON object:
{ bg, border, glow, iconColor, countColor, overlay? }

AI generates the theme object, not the icon. Ship unlimited seasonal themes without touching the icon system.
