# Glossary

- **PC (Projected Closings / Projected Commission)**: forward-looking modeled production value.
- **GCI (Gross Commission Income)**: realized or tracked commission income.
- **Actual GCI**: confirmed, realized GCI.
- **GP/VP**: pipeline stage metrics used as indicators; not closed production.
- **Pipeline Anchors**: defined leading-input variables used to ground forecasting.
- **Forecast Confidence**: confidence layer that affects display/interpretation, not base values.
- **KPI**: key performance indicator used for team/agent measurement.
- **Source of Truth**: canonical location for authoritative definition and current state.
